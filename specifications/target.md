# Target

The Target object represents the intermediate control objectives generated by the Computer.

## Structure

### pitch

The target pitch angle in degrees.

### roll

The target bank angle in degrees.

### throttle

The target throttle position (0.0 to 1.0).

### verticalSpeed

The target vertical speed in feet per minute.

## Behavior

No required behavior.

## Concurrency

This object is used by multiple threads, so it must be **immutable**.
