# Settings

This object contains all configuration for our controllers.

# State

**roll**: The configuration for the roll controller. (Configuration, SURFACE)
**pitch**: The configuration for the pitch controller. (Configuration, SURFACE)
**yaw**: The configuration for the yaw controller. (Configuration, SURFACE)
**throttle**: The configuration for the throttle controller. (Configuration, THROTTLE)

# Behavior

This object does not have any behavior.

# Concurrency

This object should be thread-safe. The references should be atomic.
